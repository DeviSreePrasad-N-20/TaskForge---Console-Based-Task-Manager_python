# TaskForge - Console-Based Task Manager

**TaskForge** is a simple, single-file Python console application to help you organize and manage your tasks straight from the terminal.

## Features

- Manage tasks with id, title, priority, due date, and status
- Add, view, update, mark complete, and delete tasks
- Filter tasks (by status, due today, due this week)
- All data saved in `tasks.json` for persistence
- No external libraries required
- Pretty-printed console tables

## How to Run

1. Make sure you have Python 3 installed.
2. Download TaskForge.py and place it in your desired folder.
3. Open your terminal in that folder.
4. Run:
5. Use the menu to add, view, update, filter, and manage your tasks.

## Data File

All tasks are stored in `tasks.json` in the same directory. The app loads your tasks at startup and saves on changes or exit.
